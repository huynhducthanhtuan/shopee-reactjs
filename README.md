### How to run this project?

#### 1. At folder you want save this project, open CMD

#### 2. Enter "git clone https://github.com/huynhdthanhtuan/shopee-reactjs"

#### 3. Enter "cd shopee-reactjs"

#### 4. Enter "npm install"

#### 5. Enter "npm start"
