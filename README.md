### How to run this project?

### 1. Clone project

```bash
$ git clone https://github.com/huynhdthanhtuan/shopee-reactjs.git
```

### 2. Change directory to folder backend-node

```bash
$ cd shopee-reactjs
```

### 3. Install dependencies

```bash
$ npm install
```

### 4. Run in developement mode

```bash
$ npm start
```
